# SQL Exploratory Data Analysis Project

## Project Description
Performed exploratory data analysis on cleaned layoffs data using SQL aggregation, filtering, and ranking functions to identify trends and insights related to layoffs by company, industry, year, and funding stage.

## Key Features
- Aggregated layoff totals and percentages by company, industry, and year.
- Created rolling totals and rankings to reveal top companies and monthly trends.
- Utilized conditional filtering to focus analysis on significant layoffs.
- Grouped data by multiple business dimensions for detailed segmentation.

## Skills Used
- Advanced SQL Aggregations (SUM, AVG, RANK)
- GROUP BY, ORDER BY, Window Functions
- Conditional Filtering
- Date/Time Data Manipulation

## How to Use
- Run the provided SQL queries after loading the cleaned layoffs dataset.
- Modify filter conditions and aggregation levels to explore different data perspectives.
